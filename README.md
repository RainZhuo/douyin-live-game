# 🎮 抖音直播间互动猜谜游戏

一个接入真实抖音直播间的弹幕互动猜谜小游戏，适合直播连麦、互动娱乐场景。

## 架构

```
┌─────────────────┐     WebSocket      ┌──────────────────────┐
│  抖音直播间      │──────────────────▶│  douyin-bridge.js    │
│  (live-parser)   │                    │  (Node.js 桥接服务)  │
└─────────────────┘                    │                      │
                                        │  - 连接抖音 WS          │
                                        │  - 解析弹幕/礼物消息    │
                                        │  - WebSocket 广播       │
                                        └──────────┬───────────┘
                                                   │ ws://localhost:6789
                                                   ▼
                                        ┌──────────────────────┐
                                        │  game-overlay.html   │
                                        │  (OBS 浏览器源)       │
                                        │                      │
                                        │  - 显示题目/提示/聊天  │
                                        │  - 处理真实+模拟弹幕   │
                                        │  - 段位/排行榜系统     │
                                        └──────────────────────┘
```

## 快速开始

### 1. 获取直播间ID

打开抖音直播网页版 `https://live.douyin.com/`，进入你的直播间，URL最后面那串数字就是直播间ID。

例如 `https://live.douyin.com/123456789` → 直播间ID: `123456789`

### 2. 启动服务

#### macOS / Linux
```bash
cd /Users/zhuo/douyin-game
./start.sh <直播间ID>
```

或直接：
```bash
cd /Users/zhuo/douyin-game
node douyin-bridge.js <直播间ID>
```

#### Windows
```batch
cd 项目目录
start.bat <直播间ID>
```

或直接：
```batch
cd 项目目录
node douyin-bridge.js <直播间ID>
```

### 3. 安装依赖（首次运行）

#### macOS / Linux
```bash
# Node.js 依赖
npm install

# Python 语音引擎
pip3 install edge-tts
```

#### Windows
```batch
:: Node.js 依赖
npm install

:: Python 语音引擎
pip install edge-tts
```

> Windows 启动脚本 (`start.bat`) 会自动检测并安装缺失的依赖。

### 4. 在 OBS 中添加

添加「浏览器源」：
- URL: `http://localhost:3000`
- 宽度: 500
- 高度: 800
- 勾选「控制音频」

## 游戏机制

### 出题系统
- 50+ 题目，8个分类（历史人物、城市、美食、动物、影视、游戏、歌曲、成语）
- 每道题5分钟倒计时，无人答对自动切换
- 上一题答案展示

### 段位系统
| 段位 | 经验分 |
|------|--------|
| 黑铁 | 0+     |
| 白银 | 100+   |
| 黄金 | 300+   |
| 铂金 | 600+   |
| 钻石 | 1000+  |
| 大师 | 1600+  |
| 王者 | 2400+  |
| 最强王者 | 3500+ |

### 礼物提示（观众在直播间送出礼物自动触发）
| 礼物 | 效果 |
|------|------|
| 📢 人气票 | 显示答案字数 |
| 🍺 啤酒 | 显示线索描述 |
| 🍭 棒棒糖 | 显示首字 |
| 🐰 比心兔兔 | 跳过本题 |

### 相似度算法
综合加权：字符集重叠(20%) + 双字组(20%) + 位置匹配(25%) + 长度比(10%) + 最长公共子串(25%)

## 手动操作

游戏页面上的按钮：
- 「暂停/恢复模拟」- 控制假玩家自动发送弹幕（用于演示/测试）
- 「下一题」- 手动切换题目
- 礼物按钮 - 点击模拟触发提示（测试用）

连接栏：
- 输入直播间ID后点击「连接」可手动连接/重连
- 连接状态显示在顶部（绿点=已连接，黄点=连接中，红点=失败）

## 注意事项

1. **必须开播才能连接** - 直播间未开播时 live-parser 无法连接
2. **抖音接口可能有变动** - 如果连接失败，可能需要更新 `@liou666/live-parser-core`
3. **礼物名称匹配** - GIFT_MAP 中的礼物名称可能需要根据实际情况调整（在 douyin-bridge.js 中修改）
4. **本服务只在本地运行** - OBS 和桥接服务在同一台机器上
